Structured-based Pharmacophores

Structured-based pharmacophore models are derived from the 3D structure of a protein. Tipically, protein-ligand complexes are used to obtain pharmacophore models, although the apo structure of the protein can also be used, when there are no known ligands. In the former pharmacoporic points are elucidated from protein-ligand interactions, while the latter employ molecular interaction fields to identify pharmcophoric points [1, 2].

Currently, OpenPharmacophore only supports pharmacophore from protein-ligand complexes, in which the pharmacophoric points are obtained from protein-ligand interactions.

Structured based pharmacophores are obtained from Protein Data Bank structure files (pdb files). To create an strucrure-based pharmacophore in OpenPharmacophore , a file can be downloaded manually from the PDB webiste, or alternatively, a PDB id can be used.

In this notebook we will obtain structured based pharmacophores for four different protein-ligand complexes.

[1]:

from openpharmacophore import StructuredBasedPharmacophore

Pharmacophore of Rhinovirus coat protein

The rhinovirus coat protein has the PDB id 1NCR and the protein is complexed with pleconaril, an antiviral compound.

To obtain the pharmacophore model we will use the StructureBasedPharmacophore class. Just as the LiganBasedPharmacophore class, the StructuredBasedPharmacophore class is derived from the Pharmacophore class. This three classes function in a very similar way.

We will use the class method from_pdb. This expects a pdb file or pdb id, the tolerance radius of the pharmacophoric points and optionally a ligand id. In case the protein contains more than one ligand, a ligand id can be passed so the pharmacohphore model is derived just for that ligand

[2]:

pharmacophore_1 = StructuredBasedPharmacophore().from_pdb("1NCR",
                                                          radius=1.0,
                                                          ligand_id="W11:A:7001",
                                                          hydrophobics="plip")
